# 🎋 古代人生模拟器

> 一个让你体验古代人生的文字养成游戏

---

## 🎯 游戏简介

从呱呱坠地到白发苍苍，你的人生由你自己选择。体质、颜值、才学、财富……各项数值将决定你的命运走向。是当官、从军、经商，还是成为名动一时的花魁？一切皆有可能。

---

## 🎮 玩法亮点

- **真实人生模拟**：从童年到老年，每个阶段都有不同的事件与挑战
- **自由选择**：结婚、单身、同性婚姻、领养子嗣，全凭你心意
- **职业多样**：朝堂为官、沙场从军、市井经商、风月花魁……多条人生路线
- **因果关联**：体质太低容易生病，颜值会随年龄衰减，每个选择都有后果
- **高度自由**：同性婚姻可能被谅解，也可能被逐出家门，命运在你手中

---

## 🖱️ 如何操作

点击屏幕上的选项，做出你的人生选择。每一次点击，都可能改变你的一生。
